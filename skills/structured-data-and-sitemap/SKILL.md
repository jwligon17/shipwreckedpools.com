# Goal
Help search engines understand page purpose, business identity, and canonical structure.

# Priorities
1. Check canonical tags first.
2. Ensure sitemap only lists canonical URLs.
3. Add or improve JSON-LD that matches visible content.
4. Validate schema output.
5. Keep robots rules from blocking important routes.

# Guardrails
- Do not add self-serving review rich-result markup for the business itself.
- Do not rely on FAQ rich-result visibility.
- Do not output structured data for content that users cannot see.
