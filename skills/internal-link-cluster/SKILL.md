# Goal
Strengthen topical clusters and help Google understand which pages matter most.

# Rules
- Use descriptive anchor text.
- Prefer contextual links over generic buttons when possible.
- Every money page should receive links from at least one hub and at least two relevant contextual sources.
- Every location page should link to its best-matching service pages.
- Every service page should link back to top-priority locations.

# Guardrails
- Do not over-link every page to every other page.
- Do not use vague anchors like 'learn more' where descriptive anchors fit.
