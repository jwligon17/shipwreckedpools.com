## Brand color direction
Use Shipwrecked Pools brand colors with:
- primary colors: navy and white
- secondary accent colors: light blue and pink

Color usage rules:
- Navy is the anchor color for header, footer, hero overlays, strong headings, and primary buttons
- White should be the dominant background/surface color across most of the site
- Light blue should be used for soft fills, subtle gradients, borders, hover states, icon backgrounds, and water-themed accents
- Pink should be used sparingly for emphasis only: eyebrow labels, small badges, active indicators, focus rings, subtle button accents, link underlines, and selective highlights
- Avoid a monochromatic all-blue look
- Keep the site bright, premium, coastal, and polished rather than dark or overly glassy
- Use more white-space contrast between sections
- Pink should never dominate large surfaces

Implementation guidance:
- create centralized theme tokens for all colors
- use CSS variables or Tailwind theme tokens so colors can be adjusted globally later
- keep good contrast and accessibility throughout

Starter palette for development only unless exact brand hex values are available:
- navy: #0B1E4B
- white: #FFFFFF
- light-blue: #A9DDF5
- pink: #E6B4C7
- dark-text-on-light: #122033
- soft-border: rgba(11, 30, 75, 0.12)
