# SEO 12 — Contact page conversion cleanup

**Suggested skill:** `contact-page-conversion`  
**Suggested agent:** `seo_copy_editor`

## Goal
Turn the contact page into a stronger quote page and remove staging language.

## Scope
Contact page copy and adjacent support elements only.

## Instructions to Codex

1. Remove the sentence `This form is staging-safe.` from the live contact page.
2. Upgrade the quote page copy to clarify:
   - what information to send
   - how Shipwrecked responds
   - that call/text is available now
   - service area language
3. If layout supports it, add a compact trust-support block near the form:
   - money-back guarantee
   - 5.0 Google rating mention
   - local-service language
4. Do not change form wiring unless the task requires it.


## Done when

- staging language is gone
- contact page reads like a live quote page
- form path and call/text path remain visible


## Constraints

- do not change form submission behavior unless necessary
